# Рутубочист

**Рутубочист** — userscript для AdGuard/Tampermonkey, который делает RUTUBE более пригодным для просмотра.

Скрипт добавляет пользовательский контроль над лентой и страницей просмотра: позволяет скрывать нежелательные каналы, Shorts, рекомендации под видео, комментарии, телевизионные разделы, промо-блоки и лишние элементы бокового меню.

## Зачем это нужно

RUTUBE можно смотреть.
Он в белых списках. Отлично подходит вместо торрентов. там есть фильмы и сериалы
Но вокруг видео слишком много того, что смотреть никто не хочет и не просил.

### Рутубочист помогает:

убрать лишнее из ленты;
скрывать нежелательные каналы;
отфильтровать видео по словам и фразам;
спрятать Shorts;
очистить страницу просмотра;
убрать рекомендации, комментарии и лишнюю обвязку;
бороться с баннерами в плеере;
сделать мобильный просмотр удобнее.

Без регистрации, без сервера, без сбора данных.
Скрипт работает только в браузере пользователя.

---

## Что умеет

### Лента и рекомендации

* скрывает нежелательные каналы;
* добавляет кнопку `⊘` для быстрого скрытия канала;
* фильтрует видео по словам и фразам;
* скрывает Shorts;
* чистит боковое меню;
* убирает телевизионные разделы, промо и саморекламу RUTUBE;
* скрывает навязанные событийные ссылки вроде «ЧМ-2026».

### Страница просмотра

* включает режим чистого просмотра;
* скрывает рекомендации справа и под видео;
* может скрывать комментарии;
* отдельно может скрывать название, описание и информацию о видео;
* препятствует автовоспроизведению видео.

### Плеер

* пытается убирать рекламные вставки в плеере;
* скрывает угловые баннеры поверх видео;
* включает системное меню браузера по правой кнопке мыши.

### Мобильный просмотр

В версии 1.3 добавлена экспериментальная свайп-громкость видео:

* горизонтальный свайп по экрану меняет громкость видео;
* слева направо — громче;
* справа налево — тише;
* громкость запоминается;
* изменение показывается поверх видео.

## «Что посмотреть»

В Рутубочист встроен раздел **«Что посмотреть»**.

Это подборки сериалов и фильмов появившихся в сети.
Подборки могут обновляться из GitHub и кэшируются локально. (т.е. если загружены, то доступны при белых списках)
Суть подборок- прокинуть поисковый запрос. не факт что видео в текущий момент времени есть на Rutube? но так найти интересное намного проще.

---

## Настройки

В панели Рутубочиста можно включать и выключать отдельные функции:

* чистка ленты и меню;
* скрытие Shorts;
* чистый просмотр;
* скрытие рекомендаций;
* скрытие комментариев;
* скрытие информации о видео;
* борьба с рекламой в плеере;
* системное меню по правой кнопке;
* свайп-громкость на мобильном;
* импорт и экспорт списков.

Списки скрытых каналов и слов можно редактировать вручную.

---

## Импорт и экспорт

Рутубочист умеет экспортировать и импортировать настройки и чёрные списки.

---

## Приватность

Рутубочист:

* не требует регистрации;
* не отправляет данные на сервер;
* не читает аккаунт RUTUBE;
* не собирает статистику просмотров;
* не следит за пользователем;
* хранит настройки локально в браузере.

Внешние запросы используются только для обновления встроенных подборок «Что посмотреть».

## Установка

Скрипт можно установить через AdGuard, Tampermonkey или другой менеджер userscript-ов.

### Установка в AdGuard

1. Откройте **Настройки → Расширения**.
2. Нажмите **Добавить**.
3. Выберите **Импортировать из файла или URL**.
4. Вставьте ссылку:

```text
https://npekpacho.github.io/rutubochist/rutube_sans_tv_adguard.user.js
```

5. Подтвердите установку.
6. Обновите страницу RUTUBE.

Если скрипт установлен по URL, он сможет обновляться с GitHub автоматически при изменении версии в metadata-блоке `@version`.
